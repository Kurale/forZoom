# Приложение для тренировки словарных слов

## Описание
Интерактивное веб-приложение для тренировки и изучения словарных слов для учащихся 3-4 классов.

## Функциональность
- Выбор уровня сложности (3 класс 1/2 часть, 4 класс 1/2 часть)
- Интерактивная игра с перетаскиванием слогов
- Проверка правильности составления слов
- Анимированная обратная связь (зелёный/красный цвет, встряска при ошибке)
- Проговаривание слов через Web Speech API
- Статистика результатов

## Как запустить

### Вариант 1: Простой запуск (рекомендуется)
Поскольку это статическое приложение, достаточно открыть файл `index.html` в браузере:

```bash
# Откройте в браузере
open index.html
# или
double-click index.html
```

### Вариант 2: Локальный сервер
Если нужен локальный сервер для CORS:

```bash
# Python 3
python3 -m http.server 8000

# Node.js (если установлен http-server)
npx http-server

# Затем откройте http://localhost:8000 в браузере
```

## Структура файлов
- `index.html` - основная HTML-структура
- `styles.css` - стили приложения
- `script.js` - логика приложения
- `words.json` - база словарных слов

## Особенности
- Адаптивный дизайн для мобильных устройств
- Поддержка drag-and-drop
- Анимации и визуальные эффекты
- Звуковое сопровождение (произношение слов)
- Полная работа оффлайн после загрузки

## Технологии
- HTML5
